## ğŸ¨ Visual Design Standards

### **Color Usage:**
- **Primary Color:** Used for CTAs, highlights (60% of CTAs)
- **Secondary Color:** Used for accents (30% of elements)
- **Neutral:** Gray scale for text, backgrounds (90% of content)
- **Accent:** Strategic use for emphasis (10% of elements)

### **Typography Hierarchy:**
- **H1:** Hero headline (48-72px)
- **H2:** Section titles (32-48px)
- **H3:** Subsection titles (24-32px)
- **Body:** 16-18px base
- **Small:** 14px for captions

### **Spacing:**
- **Section Padding:** 80-120px vertical
- **Element Gaps:** 24-40px between cards
- **Content Width:** Max 1200px (container)

### **Imagery:**
- **Hero:** Full-width, high-quality (1920x1080+)
- **Cards:** 400x300px minimum
- **Portfolio:** 800x600px minimum
- **All images:** Optimized, WebP format

---

## ğŸ“± Mobile Standards

### **Hero Section:**
- Stack vertically on mobile
- Single CTA prominent
- Reduce text size
- Full-width image

### **Cards:**
- Single column on mobile
- Touch-friendly (min 44x44px)
- Readable text (16px+)

### **Forms:**
- Stack all fields
- Large input fields
- Native keyboard types
- Clear validation

## ğŸ”„ Conversion Optimization Standards

### **CTA Placement:**
- Above fold (hero)
- After each major section
- Sticky header CTA (optional)
- Footer CTA

### **Form Optimization:**
- **Short Forms:** 3-5 fields max
- **Long Forms:** 7+ fields (for detailed quotes)
- **Progressive Disclosure:** Show fields as needed
- **Auto-fill:** Enable browser auto-fill
- **Validation:** Real-time, clear errors

### **Trust Indicators:**
- Visible in hero
- Throughout page
- In footer
- On forms
